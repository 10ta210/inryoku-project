## B. ジェネレーティブアート × POD — 全品一点物の技術

### mmERCH のパイプライン
- Seed $6.4M調達。960着の1-of-1フーディ
- フーディを6パーツに分解 → アルゴリズムが10種のプリントとレアリティを組み合わせ → NFTミント → 物理版製造
- NFCチップで「Wearable Wallet」が起動
- カシミア版はイタリアで手染め、フレンチテリー版はLAで手裁断

### 空白地帯: Three.js → POD 直結は誰もやってない
- fxhash、Art Blocks = NFTプラットフォーム（物理POD非対応）
- mmERCH = 高級マニュファクチャリング（PODではない）
- **アルゴリズミックアート → API → Gelato の自動パイプラインは未開拓市場**

### 技術実装（inryokuで実現可能）
```
[注文] → [seedからGLSLデザイン生成] → [Puppeteer + Three.jsで高解像度PNG出力]
→ [S3/CDNにアップロード] → [Gelato APIにURL送信] → [印刷・発送]
```
- Puppeteer（Headless Chrome）で既存のThree.js/GLSLシェーダーをそのまま利用
- 300DPI、3600px x 4800px のPNG出力
- seed番号をフーディにプリント → Webでseed入力 → 同じビジュアルが再現される体験

### 価格設定
| カテゴリ | 価格帯 |
|---------|--------|
| 通常PODフーディ | $25-45 |
| リミテッド（100部） | $50-150（+200-300%） |
| 1-of-1 ジェネレーティブ | $150-500+（+500%以上） |

### 著作権
- アルゴリズム自体 = プログラムとして著作権保護対象
- 人間がパラメータ設計・配色・シェーダー設計 = 「創作的寄与」→ 著作権主張可能
- seedベースの再現性 = 作品の固定性が証明可能
