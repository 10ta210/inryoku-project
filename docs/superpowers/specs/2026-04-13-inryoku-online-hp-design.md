# inryoku.online 母艦HP 設計書

## 概要

株式会社インリョクの事業サイト。3つの収入源（サイト制作・飲食店集客代行・ナレッジ販売）を1つのブランドで束ねる母艦HP。

- **ドメイン:** inryoku.online（ムームードメイン所有済み）
- **ホスティング:** Vercel（無料プラン）
- **技術:** Next.js 14 (App Router) + Tailwind CSS + TypeScript
- **デザイントーン:** ビジネス寄り・洗練・ダーク基調・ミニマル
- **レスポンシブ:** PC / タブレット / スマホ対応
- **問い合わせ:** LINE公式アカウント + メールフォーム

## ブランド定義

- **キャッチコピー:** 「見えない魅力を、引力に変える。」
- **サブコピー:** AI × デザイン × 心理学で、ビジネスの"引力"を設計する
- **カラー:**
  - Primary: #0A0A0A（ほぼ黒）
  - Accent: #FF6B35（オレンジ — CTAや強調）
  - Text: #FAFAFA（白に近いグレー）
  - Muted: #6B7280（グレー — サブテキスト）
  - Surface: #1A1A1A（カード背景）
- **フォント:** Noto Sans JP（本文）+ Inter（英字・数字）
- **ロゴ:** 既存の inryoku ロゴを使用

### 2. AI爆速サイト制作 `/web-design`

**ターゲット:** 個人事業主・店舗・スタートアップ

**構成:**
1. ヒーロー: 「最短1日。あなたのビジネスに、引力を。」
2. 特徴3つ:
   - AI × デザインで爆速制作
   - 納得いくまで修正対応
   - レスポンシブ対応・SEO基本設計込み
3. 料金: 5万円〜（LP / ブランドサイト / コーポレート）
4. 制作の流れ: ヒアリング → デザイン → 納品（最短1日）

### 3. 飲食店集客システム `/restaurant`

**ターゲット:** 飲食店オーナー

**構成:**
1. ヒーロー: 「一見さんを、あなたの店の常連客へ。」
2. 課題提示: 探されない / 選ばれない / 忘れられる
3. 解決策: MEO × LP × LINE の三段構え
4. プロセス: 認知（MEO）→ 教育（LP）→ 定着（LINE）
5. 料金プラン（3つ）:
   - ライトプラン 29,800円/月 — MEO + 口コミ促進
   - スタンダード 49,800円/月 — MEO + LP + LINE
   - プレミアム 79,800円/月 — フルサポート + 分析レポート
6. CTA → /contact

### 6. 会社概要 `/about`

**構成:**
1. ヒーロー: 「引力の源。」

   - 「見えないものを可視化する」という哲学

3. 会社情報テーブル:

   - 代表: （司さんの名前）

### 7. お問い合わせ `/contact`

**構成:**
1. ヒーロー: 「まずは、話してみませんか。」

   - LINE公式アカウント（QRコード + ボタン）
   - メールフォーム（名前 / メール / 相談内容 / 送信ボタン）
3. フォーム送信先: メール通知（Vercel Functions or 外部サービス）

## 技術仕様

### スタック
- **フレームワーク:** Next.js 14 (App Router)
- **スタイル:** Tailwind CSS
- **言語:** TypeScript
- **ホスティング:** Vercel
- **フォーム:** Web3Forms（無料・サーバーレス）or Formspree
- **アニメーション:** Framer Motion（控えめに）
- **フォント:** Google Fonts（Noto Sans JP + Inter）
