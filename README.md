# inryokü

見えないものの可視化 — 哲学をまとう服。

## Phases
- **P0**: 初代Macダイアログ + 6色球の起動シーケンス
- **P1**: Win95ローディング (ATTRACT → WARP_GROW → 101%)
- **P2**: コードワールド (量子パーティクル + 陰陽/RGBCMY球)
- **P3**: パーティクルユニバース + ECショップ (3D商品棚・OSテーマ切替)

## Stack
- Three.js 0.160.0 / GLSL / Web Audio API
- Node.js + Express (server.js)
- Stripe Checkout + Gelato (print-on-demand)

## Run
```bash
npm install
cp .env.example .env  # fill in keys
npm start             # localhost:3000
```

## Flow
Desktop: P0 → P1 → P2 → P3
Mobile: P3 直行
